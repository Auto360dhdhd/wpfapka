# MediWait

- **Evidence a SQLite databáze:** Uživatel má přehled plánovaných i proběhlých návštěv, léků a poznámek v lokální SQLite databázi.
- **MVVM a vícestránkové rozhraní:** Aplikace používá MVVM, tabové stránky (Dnešní přehled, Kalendář, Historie), commandy s parametry a převodník pro barevný stav termínů.
- **Senzory a prostředí čekárny:** Akcelerometr spouští nouzové kontakty při zatřesení a světelný senzor přepíná motiv pro lepší čitelnost v různém osvětlení.
